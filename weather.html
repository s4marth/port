API based weather app

OpenWeatherMap offers a free API for real-time weather data that you can use in your development projects

A weather based application for real time weather information retrieval on maps with additional functionalities using Openweathermap API.

API of Openweathermap were used in this project, the UI is built on html, css, react js and javascript. it was a great experience and learning behing this project.

The app basically tells you about the weather report about the any city you want to search.

Implemented and designed the webapplication completely from scratch.

Tech stack used:
  • - Html/css
  • - React js
  • - Javascript
  • - ES5/6
  • - vscode
